REQ-003 Comprobar Registro

La opción de valorar un episodio solo está disponible si el usuario está registrado y ha visualizado el episodio.

Examples

El cliente está registrado si existe un nombre de usuario guardado en el sistema. En este caso el usuario con nombre "nuevo" pasará el test. Cualquier otra combinación será fallida.

Username Sucess
john no
admin no
nuevo yes
