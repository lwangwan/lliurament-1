REQ-002 Comprobar IP

Se comprueba la direción IP de un cliente. Si está dentro del rango permitido, puede ver el episodio. En caso contrario, no se le permite visualizarlo.

Examples

El cliente tiene una dirección IP asociada. En este caso el usuario con ip "10.133.32.49" pasará el test. Cualquier otra combinación será fallida.

IP Sucess
10.133.32.50 no
10.133.42.49 no
10.133.32.49 yes
